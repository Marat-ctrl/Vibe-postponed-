# Vibe 🌊

Социальная сеть + мессенджер для iOS. Аналог Twitter с чистым дизайном и тёмными темами.

## Стек
- Swift / SwiftUI
- Firebase (Auth, Firestore, Storage)
- Xcode 15+

## Функции
- Регистрация и авторизация
- Лента постов с лайками и комментариями
- Подписки на пользователей
- Личные сообщения в реальном времени
- Поиск пользователей
- Несколько тем оформления (Dark, Vibe, White, Aqua)
- Редактирование профиля

## Запуск
1. Клонируй репозиторий
2. Добавь свой `GoogleService-Info.plist` от Firebase
3. Открой `Vibe.xcodeproj` в Xcode
4. Запусти на симуляторе или устройстве
